경쟁사 검색결과 분석

검색결과 URL을 넣으면 상위 상품 썸네일의 색감·배경 톤·할인 뱃지 패턴을 분석해 적합한 컨셉을 추천합니다. URL 도메인으로 쿠팡 / 11번가 / 네이버 쇼핑을 자동 감지하며, robots.txt를 존중하고 정적 HTML만 분석합니다.

⚠ 현재(2026-05) 세 사이트 모두 정적 HTML 수집이 막혀 있습니다 — 쿠팡: 봇 차단(403), 네이버: robots.txt 전면 금지, 11번가: 검색결과가 JS로 렌더(정적 HTML에 상품 없음). 실데이터가 필요하면 README의 쿠팡 파트너스 API 안내를 참고하세요. 로직은 정상이며 사이트가 열리면 즉시 동작합니다.

※ 같은 URL은 24시간 동안 캐시되어 재요청 시 네트워크를 다시 타지 않습니다.
